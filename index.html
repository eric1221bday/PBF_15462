Project Proposal

Proposing to create a real time fluid simulator using Position Based techniques and generating mesh using marching cubes algorithm. The final result ideally would be similar to the results of the original paper by Macklin et. al. Shown below is their result

Steps

  1. Implement Position Based Fluid simulator using algorithm described in Macklin et. al..
  2. Implement Marching Cubes Algorithm to generate fluid mesh
  3. Implement OpenGL code for real time viewing and interaction
